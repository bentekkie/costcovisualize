# Costco Purchase History Visualizer

View your Costco purchase history in an interactive and visually appealing way.

## Usage

1. Download your Costco purchase history from Costco's website, follow instructions [here](https://github.com/ankurdave/beancount_import_sources/blob/main/download/download_costco_receipts.js). If you are not in the us you should go to  https://www.costco.{your country tld}/OrderStatusCmd
2. Upload the file to the [app](http://bentekkie.github.io/costcovisualize). All data will be stored in your browser, there is no server.
3. Select the date range you want to view.
4. Explore the visualizations and insights.
